01 — Problemática

El yeso oculta lo que
el ojo no puede ver.

En el sector ganadero mexicano, las fracturas y lesiones externas en animales de alto valor genético presentan complicaciones post-quirúrgicas que frecuentemente no son detectadas a tiempo. Los métodos de inmovilización rígida como el yeso impiden la observación directa del miembro afectado.

Esta falta de visibilidad clínica permite que procesos infecciosos avancen hasta convertirse en sepsis, representando pérdidas económicas de entre $50,000 y $500,000 MXN por ejemplar, sin contar la pérdida irreversible de capital genético.

"El uso del yeso presenta la desventaja crítica de impedir la observación directa del miembro, ocultando la evolución de heridas." — Berasain, 2017
🦠

Sepsis oculta

Las infecciones progresan sin detectarse bajo el material de inmovilización.

📉

Pérdida económica

Diagnósticos tardíos en post-operatorios elevan los costos de recuperación.

🧬

Capital genético

Cada ejemplar perdido representa años de selección y mejora genética irreemplazables.

📡

Sin conectividad

02 — El Sistema

Cómo funciona
VetSense

01
🩺

Sensado

El módulo DS18B20 y AD8232 integrado en el vendaje captura temperatura local y ritmo cardiaco (BPM) del animal en tiempo real.

→
02
📶

Transmisión LoRa

Los datos se empaquetan en JSON y se transmiten via LoRa 433 MHz al gateway central. Sin WiFi, sin SIM. Hasta 2 km de alcance.

→
03
📱

Visualización

El veterinario recibe alertas en tiempo real en su dispositivo móvil vía Bluetooth desde el gateway receptor ESP32.

→
04
🚨

Alerta crítica

Si la temperatura o BPM superan los umbrales clínicos, se genera una alerta visual inmediata. Tiempo de respuesta < 10 segundos.

Ejemplo de trama de datos transmitida
|T:37.5|H:50|B:85|#1
T = Temperatura °C H = Humedad % B = BPM cardíacos
03 — Tecnología

Hardware & Arquitectura

Microcontrolador

ESP32 LoRa V3

Heltec · SX1262

Núcleo dual con Wi-Fi, Bluetooth y transceptor LoRa integrado. Opera en banda 433 MHz para máximo alcance rural.

WiFi + BT LoRa 433 MHz OLED 0.96"
Temperatura

DS18B20

Dallas / Maxim

Sensor digital de temperatura de alta precisión, lectura mediante protocolo One-Wire. Ideal para integración en yeso.

±0.5°C One-Wire 3V–5.5V
Pulso Cardíaco

AD8232 ECG

SparkFun · Analog Devices

Módulo de monitoreo del ritmo cardiaco (BPM). Permite detectar estrés o inflamación sistémica en el ejemplar.

ECG / BPM Analógico 3.3V
Protocolo

LoRa + RadioLib

Semtech · RadioLib.h

Protocolo de comunicación de largo alcance y bajo consumo. La librería RadioLib gestiona el estado del transceptor sin depender de WiFi.

2 km alcance Bajo consumo JSON payload

04 — Resultados

Lo que logramos
en la primera iteración

01

Captura estable de temperatura (DS18B20) y ritmo cardiaco (AD8232) de forma simultánea, sin conflictos de direcciones ni caídas de voltaje en el nodo sensor.

02

Comunicación inalámbrica LoRa funcional entre el módulo lector y el receptor, transmitiendo tramas JSON con temperatura, humedad y BPM cada ciclo.

03

Visualización en pantalla OLED del nodo sensor y en monitor serial del gateway, confirmando la recepción correcta de los paquetes de datos en tiempo real.

04

Integración Bluetooth exitosa: los datos se despliegan en la aplicación nRF Toolbox en dispositivo móvil, validando la cadena completa sensor → LoRa → BLE → móvil.

05

Los cambios térmicos en el ambiente son detectados y reportados en menos de 10 segundos, cumpliendo el umbral mínimo para alertas clínicas de utilidad veterinaria.
